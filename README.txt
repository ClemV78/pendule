Permet d'afficher un pendule

Utilisation :
Lancer le fichier affichage.py
La longueur du fil (l) et le pas de temps (pas) peuvent être changés dans le fichier equations.py
La durée (tf) peut être changée dans le fichier affichage.py

Autres fichiers :
equations : résout l'équation du pendule avec la méthode de Newton
affichage_essaie : affiche une balle qui rebomdit indéfiniment
